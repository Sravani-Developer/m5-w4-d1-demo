# Module 5 Week 4 Day 1 Demo

This repository contains my work for the Module 5 Week 4 Day 1 in-class DOM and React exercise.

The project includes three small demos:

- `demo1.html` shows how to create and append an `h1` element using JavaScript DOM methods.
- `demo2.html` updates the page text color, background color, and link color based on dropdown selections.
- `demo3/index.html` uses React to display clickable elements and shows the clicked element's ID in an alert.

## Project Structure

```text
m5-w4-d1-demo/
├── demo1.html
├── demo2.html
├── demo3/
│   ├── index.html
│   ├── on-click.js
│   └── styles.css
└── README.md
```

## How to Run

Clone the repository and start a local server from the project root:

```powershell
python -m http.server 3000
```

Then open these pages in your browser:

- `http://localhost:3000/demo1.html`
- `http://localhost:3000/demo2.html`
- `http://localhost:3000/demo3/index.html`
